# 📋 Gestion des Clubs
Application complète de gestion pour clubs sportifs :
- Gestion du stock + codes-barres + étiquettes personnalisées
- Caisse & ventes fonctionnant hors ligne
- Inventaire, rapports et statistiques
- Gestion des utilisateurs et droits
- Multi-clubs + abonnements modulables
- Déploiement via Docker / Dokploy / IONOS
